## Como gravar

- **GIF de tela no Windows:** [ScreenToGif](https://www.screentogif.com/) (grátis,
  leve). Grave a região da taskbar, exporte como `.gif`.
- Mantenha os arquivos com no máximo ~5–8 MB pra carregar rápido no GitHub.
- Depois de adicionar os arquivos aqui, os caminhos no README já funcionam.
